## What worked well

- **10-slot parallelism** ‚Äî 11 opus implementations + sonnet QA/review ran concurrently with no conflicts. All batch 1+3 issues completed in ~25 minutes wall clock.
- **Adversarial review caught real bugs** ‚Äî #713 had 2 P0s (restart broken, unhandled error) that would have shipped. The repair‚ÜíQA cycle worked cleanly.
- **`--worktree` for everything** ‚Äî after the user flagged `--cwd` issues, switching all sessions to `--worktree` eliminated worktree escape problems. Bye auto-cleanup worked perfectly for `--worktree` spawned sessions.
- **Inline fixes during orchestration** ‚Äî #742 (watcher test budget violation) was blocking the release commit. Refactoring the test file directly (31‚Üí14 tests) unblocked the release without spawning a session.

## What didn't work

- **`--cwd` worktree escape** ‚Äî QA sessions spawned with `--cwd` were doing `git checkout main` to merge PRs, which pulled changes into the main repo. Root cause was missing `PWD` env var, but the QA sessions' merge behavior compounded it. Had to manually clean up `qa-*` worktrees.
- **Batch 2 never started** ‚Äî #704 (Plans tab UI, the big one) completed late and its PR needed adversarial review repair (#748, #749). The 3 dependent issues (#705, #706, #707) carried forward to sprint 14.
- **Worktree accumulation** ‚Äî 130+ old worktrees from previous sprints in `.claude/worktrees/`. No automated cleanup for stale worktrees.
- **Pre-commit hook takes ~100s** ‚Äî the sequential test profiler dominates commit time. Filed #750 for parallelization.

## Patterns established

- **PWD must be set for Bun.spawn with cwd** ‚Äî shells do `chdir() + PWD=`, Bun only does `chdir()`. This is the root cause of directory escape bugs. Now fixed in `defaultSpawn()`.
- **Release thresholds tightened** ‚Äî major=breaking top-level command, minor=new command/package, patch=everything else. Prevents version inflation from feature flags/enhancements.
- **DB retains session cost after cleanup** ‚Äî `agent_sessions` table keeps `total_cost` and `total_tokens` after sessions end. $643 total across 7 days of sessions.
- **Always `--worktree`, never `--cwd` for QA** ‚Äî until the PWD fix is validated, `--worktree` is the safe path for all session phases.
