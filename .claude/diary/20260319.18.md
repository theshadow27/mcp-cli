## What worked well

- **Third consecutive clean sweep (15/15)**: 46 PRs across 3 sprints with zero drops. The pipeline is mature.
- **ACP provider shipped in one sprint**: From validated spike to full provider + CLI commands + TUI integration in a single sprint. The spike-first approach (#518 → #519) paid off — implementers had concrete protocol knowledge.
- **Agents tab renamed cleanly**: The DB was already provider-agnostic (from good earlier design). The rename was mostly mechanical — components, hooks, tab definitions.
- **Codex found its own bug**: The Codex self-repair session identified the real protocol mismatch (envelope wrapping) through raw wire probing. We shipped its fix (#863) and filed actionable retro items.
- **Copilot collaboration**: User manually tested the ACP spike with `gh copilot --acp`, then Copilot itself refined the spike script in a worktree. Multi-provider development is working.

## What didn't work

- **#519 ACP provider cost $20+ across sessions**: Implementation ($1.17 plan + impl), adversarial review ($1.00), repair ($4.39), QA ($1.01). The plan-mode pause added latency — session waited for approval on a well-specified issue.
- **2 sessions stuck in connecting (Sprint 17 regression)**: Fixed in this sprint (#837), but cost us respawns at sprint start. The fix adds a 30s connect timeout with retry.
- **Duplicate issues continue**: Sessions independently filed similar coverage/test issues. Need to add "check existing issues" instruction to implement skill.

## Patterns established

- **Spike → implementation → CLI → TUI is the right sequence**: ACP followed this path perfectly (#518 spike → #519 provider → #520 CLI → #862 TUI). Each step validated the previous.
- **Cross-provider collaboration works**: Claude orchestrates, Claude implements, Codex debugs its own protocol, Copilot validates its own spike. Each model contributes where it's strongest.
- **Stale binary warning is live**: #864 ensures agents (and humans) notice when dist/ is outdated. Addresses the Codex retro finding.
